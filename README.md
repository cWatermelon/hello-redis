### Features
✅ delete key-value

## Develop

Cd webapp and run `pnpm run start` for webapp dev server

Cd server and run `pnpm run start:debug` for server dev

## Build

Cd webapp and run `pnpm run build`

Cd server and run `pnpm run start:prod`
